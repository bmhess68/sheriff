# Sheriff Brian M. Hess Website

A static public website for Sheriff Brian M. Hess with current Putnam County Sheriff's Office resources, community updates, and scholarship fund information.

## Run Locally

```bash
npm start
```

The site is served from the repository root at `http://localhost:8000`.

## Structure

```text
index.html
assets/
  css/style.css
  js/main.js
  images/
```

## Deployment

This is a static site with no build step. If the DigitalOcean app is connected to this GitHub repository, pushing to the configured branch should trigger the deployment.

## Content Notes

- Official Sheriff's Office links point to `https://putnamsheriff.com/`.
- Scholarship applications link to the public Jotform listed in Putnam Valley High School guidance materials.
- The site intentionally keeps a small amount of campaign-era history while presenting the office and community work as current.
